# ALL

- ボタンをホバーしたときに何をするボタンなのかの詳細を表示する
	- G-codeコマンドを送信するときはそれも表示する
- ドロップダウンメニュー UIをmac標準ではなく、アプリ内独自のものにする

# Toolbar

- ホーミングボタンと、原点復帰ボタンがわかりづらい
	- ホーミング、ロック、アンロックはTop Toolbarに追加する方針の方がいいかもしれない
- ロック、アンロックのトグルがされた時にアイコンが変化しない

# Jog Control

- Jogコントロールのステップサイズをカスタム指定できるようにして欲しい

# Console

- できるだけConsoleがGRBLからの`OK`で隠れないようにしたい
- Consoleパネルの内容を範囲選択してコピーできるようにしてほしい
- Consoleパネルの領域とビジュアライザーの領域の境を動かせるようにしたい

# Visualizer

- 実線として描かれるパスと、移動のみのパスを視覚的に区別できるようにする
- ズーム倍率を等倍（100%）にできるボタンが欲しい
- 読み込んだG-codeファイルの描画範囲領域に応じてVisualizerの範囲を調整できるボタンを追加したい
- ガイド線を任意に配置できるようにしたい
	- 原点を左下として、正の方向に用紙を配置したときの仮想上の大きさを出してほしい
		- 一通りの用紙サイズと、カスタムサイズ（W,H mm）で指定できるようにしたい
- 実行しているgcodeファイルのプログレスバーにプログレスパーセンテージを追加して欲しい
- Visualizer上で表示されるマウスカーソルの形状はmoveにする

# Setting

- ポーリングレートを10,20,50,100,200,500,1000msから選べるようにしてほしい
- Setting Drawer以外の部分をクリックした場合に、Setting Drawerを畳むようにする
## Serial Port

- Serial PortのDisconnect機能が動作していない可能性がある
	- ボタンサイズを調節する
	- Disconnectボタンに切り替わったら、ボタンの色を赤に変更する
- Serial Portのリロードボタンの追加
- Port一覧の中からArduinoに接続されているものがあれば、それを最優先にする
